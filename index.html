Welcome to the SIMON Game!

1. Press ANY button on the keyboard to start. Simon will give the first signal. Repeat the signal by clicking the same color button.

2. Simon will duplicate these first two signals and add one.

3. Continue playing as long as you can repeat each sequence of signals correctly.

4. If you fail to repeat a sequence exactly, Simon responds with a "WRONG" sound. This means you've lost, and the sequence of signals ends.

5. You will need to press any button on the keyboard to START the game again.

Press Start the Game button to Start
